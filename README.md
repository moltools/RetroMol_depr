# RetroMol

RetroMol is a Python package for retrosynthetic analysis of molecules.

## Installation

Create and activate environment:
    
    ```bash
    conda create -f environment.yml
    conda activate retromol
    ```

## Usage

    ```bash
    retromol -h
    ```
